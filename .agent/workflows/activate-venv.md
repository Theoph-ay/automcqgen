# Activate Virtual Environment

Before running any Python or pip commands in this project, always activate the virtual environment first:

// turbo-all

1. Prepend `.venv\Scripts\activate;` to any Python/pip command, e.g.:
   ```powershell
   .venv\Scripts\activate; python my_script.py
   ```
